# Archive-It Utilities

This repository contains source code for extracting information from Archive-It collections. Most work is currently done through a single class `ArchiveItCollection`, which performs screen-scraping in order to acquire general collection metadata, seed lists, and seed metadata.

# Installation

After cloning this repository, change to source code the base directory and run:
`python ./setup.py install`

or
`pip install`

The package is installed as `aiu`.
